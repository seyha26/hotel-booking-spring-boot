# Hotel Booking System (Spring Boot + Thymeleaf)

This project is created for **learning purposes**, focusing on building a web application using **Spring Boot** and **Thymeleaf**.

The backend is developed with Spring Boot, while the frontend UI is adapted from an existing Bootstrap HTML template and rendered using Thymeleaf.
